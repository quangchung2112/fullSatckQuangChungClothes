# Dự Án Fullstack

## Mô Tả
Dự án này là một ứng dụng Fullstack bao gồm một backend được xây dựng bằng Node.js, Express.js, Sequelize và MySQL (quản lý qua phpMyAdmin) và một frontend được phát triển bằng React.js.

## Cách Chạy Dự Án

### 1. Chạy Backend
1. Mở terminal và điều hướng đến thư mục backend:
cd backend
npm install
npm start

### 2. Chạy Frontend
cd frontend
npm install
npm run start


### Nhập Cơ Sở Dữ Liệu
Mở phpMyAdmin.
Tạo một cơ sở dữ liệu mới tên dataclothes.
Chọn cơ sở dữ liệu vừa tạo.
Chuyển đến tab Import.
Chọn file SQL từ thư mục my-fullstack-project/dataclothes.sql.
Nhấn nút Go để nhập dữ liệu.
